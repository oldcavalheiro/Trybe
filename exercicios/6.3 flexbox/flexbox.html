Flex

Properties for the Parent (flex container)

flex-flow == flex-direction + flex-wrap

flex-direction default(row)
define o sentido do "main axis" alterando a direção dos itens

flex-wrap nowrap(default)
(nowrap) itens em um única linha e podem exceder o limite do container
(wrap) itens que passam do container vão para próxima linha

justify-content flex-start(default)
define espaço que os elementos ocupam e o espaço entre eles pelo main-axis

align-items stretch(default)
controla o alinhamento direto de todos os itens filhos pelo Cross Axis

align-content normal(default)
property sets the distribution of space between and around content items along a flexbox's cross-axis

Properties for the Children (flex items)

  1. order
  2. flex-shrink
  3. flex-grow
  4. flex-basis
    default size of an element before the remaining space is distributed
  5. flex == flex-grow, flex-shrink and flex-basis
    default is 0 1 auto
  6. align-self
